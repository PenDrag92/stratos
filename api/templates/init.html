Welcome to the STRATOS API

Welcome to the new STRATOS API. This resource was developed by Angel Villar and Philipp in order to made possible a feasible and easy way to transfer the data
measured by the sensors in the STRATOS balloon.

This API REST service allows a user carry out queries in order to obtain information.
So far only three request methods have been implemented:
     - Requested all data
     - Requested data by parameter
     - Requested data by data
